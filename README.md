# Präsentation: Der Werkstoff Aluminium (Lernfeld 5, Nutzfahrzeugbau)

Strukturierte PowerPoint-Präsentation/Handout zum Werkstoff Aluminium,
erstellt anhand der Aufgabenstellung (Mercedes-Benz 2440 L).

## Deliverable
- `Praesentation_Aluminium.pptx` – 11 Folien, 16:9, mit Quellenangaben

## Inhalt (gemäß Aufgabenstellung)
1. Vorkommen & Abbaugebiete
2. Vor- und Nachteile
3. Herstellung (Bayer-Verfahren + Schmelzflusselektrolyse)
4. Legierungen: aushärtbar/nicht aushärtbar · Knet- vs. Gusslegierung
5. Festigkeitssteigerung (3 Verfahren)
6. Anwendung im Berufsalltag

## Reproduzieren
```
pip install python-pptx matplotlib pillow
python3 make_images.py   # erzeugt Grafiken in assets/
python3 make_pptx.py     # baut Praesentation_Aluminium.pptx
```
